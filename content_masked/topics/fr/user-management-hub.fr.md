Le **Hub de gestion des utilisateurs** affiche une vue d'ensemble de tous les opérateurs et groupes d'opérateurs dans votre compte. Depuis cet espace, vous pouvez consulter, gérer et configurer les paramètres et les droits des utilisateurs.

![Hub de gestion des utilisateurs]([LINK_URL_1])

Pour accéder à cette page, passez par le menu [SHORTCODE_1] Configuration de compte > [SHORTCODE_2] [Opérateurs et groupes]([LINK_URL_2]). Différents blocs affichent les informations suivantes :

- Le nombre total d'opérateurs inclus dans votre compte, ainsi qu'une liste des opérateurs actifs. Cliquez sur le bouton [SHORTCODE_3] Voir tous les opérateurs [SHORTCODE_4] pour ouvrir la **vue d'ensemble des opérateurs**. Depuis cette page, vous pouvez configurer les [droits et les paramètres de chaque opérateur]([LINK_URL_3]). Le bouton [SHORTCODE_5] Ajouter un opérateur [SHORTCODE_6] vous permet de [créer directement un opérateur]([LINK_URL_4]).

- Le nombre total de groupes d'opérateurs inclus dans votre compte, ainsi que les groupes n'incluant pas d'opérateurs. Cliquez sur le bouton [SHORTCODE_7] Voir tous les groupes d'opérateurs [SHORTCODE_8] pour ouvrir la **vue d'ensemble des groupes d'opérateurs**. Depuis cette page, vous pouvez configurer les [droits et les paramètres de chaque groupe d'opérateurs]([LINK_URL_5]). Le bouton [SHORTCODE_9] Ajouter un groupe d'opérateurs [SHORTCODE_10] vous permet de [créer un groupe d'opérateurs]([LINK_URL_6]).

- Le nombre total d'utilisateurs de l'API, ainsi que le nombre d'utilisateurs API inactifs depuis plus de 30 jours. Cliquez sur le bouton [SHORTCODE_11] Voir tous les utilisateurs API [SHORTCODE_12] pour ouvrir la **vue d'ensemble des utilisateurs API**. Depuis cette page, vous pouvez consulter les informations relatives aux utilisateurs de votre API, ainsi que les [informations sur l'API concernant les opérateurs individuels]([LINK_URL_7]).

- Le nombre total d'opérateurs actuellement [en service]([LINK_URL_8]).

- Le nombre total d'opérateurs créés dans votre compte. Notez que le nombre d'opérateurs peut être limité selon le type de compte.

- Le nombre total de moniteurs de production disponibles. Cliquez sur ce bloc pour afficher la [liste des moniteurs et des droits d'utilisation accordés aux opérateurs et aux groupes d'opérateurs]([LINK_URL_9]).

Par ailleurs, le menu latéral dans le **Hub de gestion des utilisateurs** contient des informations utiles et des liens vers les articles de la base de connaissances.
